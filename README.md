# chat-app
Socket IO chat app 

This app is based on the tutorial [Click](https://www.youtube.com/watch?v=-0j_9LqsOkQ&list=PLDlWc9AfQBfbyGwhSlxg16mQGpGnauCwq)

# Installation
Clone or download app 

* *cd path/to/app*
* *npm install*
* *node server.js*
